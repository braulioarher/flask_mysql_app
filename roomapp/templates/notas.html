Cargar configuranciones de Flask

En caso de que queramos ver las configuranciones de nuestra aplicacion debemos iniciar nuestra terminal de Flask usando el siguiente comando flask shell posterior

Visualizando configuraciones

>>> from pprint import pprint
>>> from project import app
>>> pprint(app)
>>> pprint(app.config)

Aqui podemos ver el estado de todas las variables de configuracion de Flask

Rutas con variables

@app.route('/user/')
def show_user_profile(username):
    # show the user profile for that user
    return f'User {escape(username)}'
@app.route('/post/')
def show_post(post_id):
    # show the post with the given id, the id is an integer
    return f'Post {post_id}'
@app.route('/path/')
def show_subpath(subpath):
    # show the subpath after /path/
    return f'Subpath {escape(subpath)}'

La ruta acepta un valor como variable y regresa dicho valor
