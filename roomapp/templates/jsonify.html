flask.jsonify

jsonify en Flask serializa los datos al tipo de respuesta JSON, y a diferencia de usar json.dumps() la funcion jsonify() contiene el apropiado content-tyoe header 'application/json para usarse con las respuestas json. Por otra parte json.dumps() solo regresa un string codifcado teniendo que anadir al header MIME type manualmente

Ejemplo de uso jsonify con make_response

@app.route('/jsonify')
def jsonify():
    dir = {'nombre': 'Braulio', 'edad': 24}
    response = make_response(jsonify(dir), 401)
    return response

La ruta regresa una respuesta con la informacion de dir y un codigo de respuesta
