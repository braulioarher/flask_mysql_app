Archivo config.py


Dentro del archivo config.py podemos establecer las configuraciones de nuetra app por ejemplo la direccion de nuestra base de datos y nuestra SECRET_KEY

Codigo.

import os
basedir = os.path.abspath(os.path.dirname(__file__))

class Config(object):
    SQLALCHEMY_DATABASE_URI = os.environ.get('DATABASE_URL') or 'sqlite:///' + os.path.join(basedir, 'hello.db')
    SQLALCHEMY_TRACK_MODIFICATIONS = False
    SECRET_KEY = os.environ.get('SECRET_KEY') or 'you-secret-key'

Como se observa se crea una clase Config la cual herdera a object dentro de esta clase podemos agregar variables de configuracion segun las vayamos necesitando, es importante que una vez configuradas todas nuestras variables de configuracion las importemos dentro de nuestro archivo __init__.py una vez importado defnimos las configuraciones dede un objeto como se muestra a continuacion


from .config import BaseConfig

app.config.from_object(BaseConfig)    #Cargamos las configuraciones de la app
