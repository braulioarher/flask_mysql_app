Crear aplicacion Flask

En este sitio podras saber como crear una aplicacion Flask y la estructura de los elementos que la componen

Requisitos.

Para crear una aplicacion en Flas previamente se debe de tener instaldo el paquete de Flask en el entorno virtual esto lo podemos lograr usando el comando pip install flask una vez instalada la paquteria podemos crear una carpeta que incluya el archivo __init__.py o bien podemos crear un archivo llamado main.py este archivo sera la base de nuestra aplicacion dento de nuestra carpeta principal se puede crear sub carpetas como templates o static las cuales ayudan a tener mas orden en nustro proyecto adicional a las carpetas se pueden crear archivos como: config.py, forms.py, models.py y routes.py archivos los cuales ayudan a contener funciones y clases que le dan funcionalidades a nuestra aplicacion

Estructura de nuestra aplicacion

foldername

| - __init__.py

| - config.py

| - forms.py

| - models.py

| - routes.py

| - templates

| - static

Creado la app

Para crear la aplicacion como se menciono ateriormente se puede hacer de dos formas bastantes similares la diefencia radica en el archivo __init__.py tenemos que crear una funcion que regrese nuestra aplicacion mientras que al usar archivo main.py debemos de crear los comando como se muestran a continuacion

from flask import Flask
from .config import BaseConfig
from flask_sqlalchemy import SQLAlchemy
from flask_migrate import Migrate

app = Flask(__name__) #Hacemos una instancia para crear la app
app.config.from_object(BaseConfig) #Cargamos las configuraciones de la app

bootstrap = Bootstrap(app)
db = SQLAlchemy(app)
migrate = Migrate(app, db)

Una vez intanciadas nuestras variables de aplicacion podemos importar nuestra app desde nuestro archivo main.py usando form foldername import app de esta forma podemos exportar nuestra variable FLASK_APP a nuestra terminar usando el comando export FLAS_APP=main.py

La siguiente manera es similar solo creado la funcion create_app y retornando la variable app

from flask import Flask
from .config import BaseConfig
from flask_bootstrap import Bootstrap
from models import db

def create_app(test_config=None)
    app = Flask(__name__) #Hacemos una instancia para crear la app
    app.config.from_object(BaseConfig) #Cargamos las configuraciones de la app

    bootstrap = Bootstrap(app)
    db.app = app
    db.init_app(app)

    return app

De esta forma entonces podemos exportar nuestra variable FLASK_APP a nuestra terminar usando el comando export FLAS_APP=foldername
